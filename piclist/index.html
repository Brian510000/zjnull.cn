基于PicList + Gitee 搭建图床

前言

如果你想在手机或电脑上轻松管理图床，PicList 是一个不错的选择。它支持多种存储方案，其中 Gitee/GitHub 是免费的最佳选择。本教程将指导你如何用 PicList + Gitee 搭建一个高效的个人图床。

GitHub 图片可能在国内加载较慢，故推荐使用 Gitee 进行加速

1. 准备工作

  • Gitee 账号（注册 Gitee）
  • Gitee 私人令牌（用于授权 PicList 上传图片）
  • PicList 应用（下载地址）

2. 创建 Gitee 仓库

  1. 登录 Gitee，点击右上角 + → 新建仓库。
  2. 设置仓库名称（如 pic-bed），保持 公开。
  3. 点击 创建 按钮完成。

3. 获取 Gitee 私人令牌

  1. 进入 账号设置 → 安全设置 → 私人令牌。
  2. 点击 生成新令牌。
  3. 选择 projects 和 repos 权限。
  4. 复制生成的令牌（只会显示一次）。

4. 配置 PicList

  1. 打开 PicList，进入 插件
  2. 选择 gitee 2.0.7 插件进行安装/或更高版本
  3. 点击插件右下角 设置 按钮，进入配置uploader页面
  4. 填写以下信息：
    • owner：用户名（你的 Gitee 用户名）
    • repo：仓库名
    • Token：粘贴 Gitee 私人令牌。

      域名：https://gitee.com/用户名/仓库名/raw/分支名/。

  5. 保存设置

5. 设为默认

  1. 点击图床，下拉至Gitee图床
  2. 点击设为默认图床
    \

6. 测试上传

  1. 进入 PicList 上传区，选择一张图片。
  2. 上传成功后，会生成 Markdown 格式的图片链接。
  3. 复制链接，在博客或文档中使用。

结语

至此，你已成功搭建 PicList + Gitee 个人图床，手机端和 PC 端均可轻松上传、管理图片。快来体验高效便捷的图床吧！
